
DDL (Data Defination Language)

1.CREATE - it is used for create table ,create database , create procedure etc.

example:
mysql> create database topic;
Query OK, 1 row affected (0.02 sec)

How to create table using a create command 

Syntax: 
	create table tablename(colname datatype(size),colname datatype(size).... Col datatype(size));

example:
mysql> create table employee(id int,name varchar(50),department varchar(200),salary int);
Query OK, 0 rows affected (0.21 sec)

2.DESC - this command is used for describe the table structure like as column name in table, column data type , column size etc 

Syntax:  DESC tablename;

example:

mysql> DESC employee;

3.ALTER COMMAND :alter command is used for modify the column name, add new column,
 remove column from table , change column type or size etc

Alter command provide following options 
I.ADD :this option helps us to add a new column in the database table after table creation  
Syntax: alter table tablename  add columnname datatype(size);
	 Or 
We can add more than one column at time using alter statement 
 Syntax: alter table tablename  add (columnname datatype(size),columnname datatype(size));
 
example: Example: we have employee table with field eid,name,email,salary and
 we want to add two more column in employee table address, contact

V.DROP STATEMENT: 
Drop statement is used for drop the table , drop or delete the database , procedure,trigger,function ,index etc 
Syntax: drop table tablename;


example:
mysql> DROP DATABASE topic;
Query OK, 0 rows affected (0.08 sec)
